Kiddo API

This API responses to Kiddo Application

new_user[POST]

create new user .

Variables:
name,
phone,
email,
department,
picture

Query format:
https://kiddo2018.herokuapp.com/user/new_user/

new_task[POST]

Create a new task, for a specific department.
Must have a logic date sumbission.

Variables:
topic,
submission,
teacher,
content,
department

Query format:
https://kiddo2018.herokuapp.com/coordinator/new_task/

-------------------------------------------------------------------------------------------------------------------------------------

task_complete[POST]

Update the status by Topic in taskList of selected user by email, completed

Variables:
t_state_id

https://kiddo2018.herokuapp.com/user/task_complete/

delete_user[DELETE]

email
